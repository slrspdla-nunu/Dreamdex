## 3. 화면별 콘텐츠 인벤토리

### 1. 정원 (홈) — 핵심
- **상단 히어로**: 날짜 · 인사 · 인라인 요약(총 꿈 N · 키운 식물 M · 다음 해금까지 K)
- **지역 탭/뷰**: 해금된 지역 전환. 각 지역은 식물이 배치된 "땅".
  - 식물 = 성장 단계별 SVG. 호버/탭 → 이름·단계·감정 미리보기.
  - 미해금 지역 = 잠금 카드(필요 꿈 수 안내).
- **빈 상태**: 묘목장에 "첫 씨앗을 심어보세요" + 예시 데이터 버튼.
- 식물 클릭 → 식물 상세.

### 2. 꿈 기록 (작성/수정)
- 제목 · 날짜 · 감정(단일) · 내용
- 저장 시: 추출 키워드 → **씨앗 생성/성장 결과** 피드백
  - "🌱 새 씨앗 3개: 학교·친구·시험" / "🌿 *학교*가 새싹이 되었어요"
  (실제 렌더는 이모지 아닌 커스텀 글리프)

### 3. 아카이브 (꿈 목록)
- 검색 · 최신순 · 감정 필터 · 즐겨찾기 (기존 유지)
- 카드: 제목·날짜·감정·발췌·키워드 칩

### 꿈 상세
- 전체 내용 · 감정 · 날짜 · 수정/삭제/즐겨찾기
- **이 꿈에서 심은 씨앗**(키워드) 칩 → 식물 상세로 이동

### 4. 도감 (Codex) — 2탭
- **꿈 도감**: 장소/인물/상황 발견 현황 (관찰, ≥1회) — 기존 구조 유지
- **식물 도감**: 재배 현황 (≥2회=새싹부터)
  - 진행: 일반 X/92 · 희귀 Y/19 · 전설 Z/9
  - 슬롯: 발견 식물(성장 단계 아이콘) / 미발견(실루엣) / 잠긴 tier(해금 안내)

### 식물 상세
- 식물명(키워드) · 형태(카테고리) · 대표 감정 · 희귀도
- **성장 단계 게이지**(현재 단계 / 다음 단계까지 N회)
- 소속 지역 · 첫 발견일 · 등장한 꿈 목록(→ 꿈 상세)

### 5. 통계 / 6. 꿈 지도 / 7. 설정
- 기존 유지. 설정에 **예시 데이터 불러오기**, 테마 3종, 백업/초기화.

## 5. 데이터 소스 (개념)

저장(LocalStorage): **`dreams[]` · `settings`** 2가지뿐. (변경 없음)
런타임 파생(저장 안 함):
- `keywordDictionary` → 분류·도감 분모 (기존)
- **`buildGarden(dreams)`** → 식물 목록(키워드별 count·대표감정·성장단계·tier·지역·등장 꿈) ★ 신규 파생
- `buildDex` / `buildStats` / `buildGraph` (기존)

> 정원·식물·성장은 모두 dreams에서 **매번 계산**. 새 스키마 불필요 → 기존 아키텍처와 일관.
