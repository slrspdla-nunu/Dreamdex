# Dream Garden — 사용자 플로우 (개정판)

핵심 경험: **꿈 → 씨앗 → 식물 → 정원**. 모든 플로우는 정원(홈)으로 수렴한다.

## 플로우 A. 첫 진입 → 첫 씨앗
```
온보딩("꿈은 씨앗이다") → [정원=홈 / 묘목장 비어있음]
   → "첫 씨앗 심기" CTA → 꿈 기록
   → 저장 → 씨앗 생성 연출(키워드 → 씨앗 N개) → 정원 묘목장에 씨앗 등장
분기: 둘러보기 → "예시 데이터 불러오기" → 정원이 즉시 풍성해짐(전 기능 시연)
```

## 플로우 B. 반복 기록 → 성장 (핵심 동기 루프)
```
꿈 기록(같은 키워드 재등장) → 저장
   → 성장 판정: 해당 키워드 누적 횟수 증가
   → 단계 상승 시 피드백 토스트:
        "🌱 새 씨앗: 바다"        (1회)
        "🌿 학교가 새싹이 되었어요" (2회 → 식물 도감 등록)
        "🌸 친구가 만개했어요"     (5회)
   → 정원에서 해당 식물 시각적으로 성장
엣지: 새 키워드 없고 성장도 없으면 → "기록했습니다"만
```

## 플로우 C. 지역 해금
```
꿈 개수 임계 도달(10/30/50/100) → 해금 연출(모달/하이라이트)
   "달빛 온실이 열렸습니다" → 해당 카테고리 식물이 새 지역으로 이주
   30/50: 희귀/전설 식물(키워드 tier) 발견 가능해짐
정원에서 새 지역 탭 활성화 → 더 넓고 풍성한 정원
```

## 플로우 D. 정원 탐색 → 식물 상세
```
정원 → 지역 탭 전환 → 식물 클릭
   → 식물 상세: [형태·대표감정·희귀도] [성장 게이지: 현재단계 / 다음까지 N회]
                [소속 지역] [등장한 꿈 목록]
   → 꿈 클릭 → 꿈 상세 → (수정 시 키워드/성장 재계산)
```

## 플로우 E. 도감 — 관찰 vs 재배
```
도감 → [꿈 도감] 탭: 본 요소(≥1) — 미발견은 ??? 실루엣
     → [식물 도감] 탭: 키운 식물(≥2=새싹~) — 일반/희귀/전설 진행률
          미발견=실루엣, 잠긴 tier=해금 안내(꿈 30/50)
          식물 클릭 → 식물 상세
```

## 플로우 F. 꿈 수정/삭제 시 정원 일관성
```
꿈 수정(내용/감정 변경) → 키워드 재분류 → buildGarden 재계산
   → 식물 성장단계·대표감정이 즉시 반영(파생 데이터라 자동)
꿈 삭제 → 해당 키워드 횟수 감소 → 식물이 단계 하락하거나 사라질 수 있음(정직한 반영)
```

## 화면 전이 요약
```
온보딩 ─→ 정원(홈) ─┬─→ 기록 ─→(씨앗/성장 피드백)─→ 정원
                    ├─→ 아카이브 ─→ 꿈 상세 ─→ 식물 상세
                    ├─→ 도감 [꿈|식물] ─→ 식물 상세 ─→ 등장 꿈
                    └─→ 탐험 ─→ 통계 / 지도
설정은 어디서나 헤더로 접근
```

## 피드백/연출 원칙
- 성장·해금 연출은 **짧고 조용하게**(별빛 반짝임, 부드러운 스케일). 과한 게임 연출 금지.
- 모든 식물/씨앗/단계 아이콘은 **커스텀 SVG**(이모지 미사용), 테마 색 토큰 사용.
- `prefers-reduced-motion` 존중.
